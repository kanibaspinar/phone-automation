Device Endpoints

GET /api/devices/list

List Available Devices

Get a list of all available (unassigned) devices and their status.

Response Example:

{
    "success": true,
    "devices": [
        {
            "id": 1,
            "device_id": "device123",
            "device_name": "P1",
            "status": "connected",
            "last_seen": "2024-01-30T12:00:00Z",
            "is_initialized": true,
            "instagram_accounts": [
                {
                    "username": "user123",
                    "login_status": true,
                    "last_login": "2024-01-30T12:00:00Z"
                }
            ]
        }
    ]
}
GET /api/devices/user/{user_id}

Get User's Devices

Get all devices assigned to a specific user.

Path Parameters:

  • user_id (string, required): The ID of the user
POST /api/devices/create

Create New Device

Register a new device in the system.

Request Body:

{
    "device_id": "device123",
    "name": "P1"
}
POST /api/devices/assign

Assign Device

Assign a device to a user.

Request Body:

{
    "user_id": "user123",
    "count": 1  // Optional: Number of devices to assign
}
DELETE /api/devices/{device_id}

Delete Device

Remove a device from the system.

Instagram Endpoints

POST /api/instagram/login

Login to Instagram

Login to an Instagram account on a specific device.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "password": "pass123",
    "email": "optional@email.com",  // Optional
    "email_password": "optional_email_pass"  // Optional
}
GET /api/instagram/accounts/list

List Instagram Accounts

Get a list of all Instagram accounts with optional filtering.

Query Parameters:

  • device_id (string, optional): Filter by device ID
  • login_status (boolean, optional): Filter by login status
  • username (string, optional): Filter by username
POST /api/instagram/actions/like-post

Like Post

Like a post from a target user.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user"
}
POST /api/instagram/actions/follow

Follow User

Follow a target Instagram user.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user"
}
POST /api/instagram/actions/unfollow

Unfollow User

Unfollow a target Instagram user.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user"
}
POST /api/instagram/actions/view-story

View Story

View a user's Instagram story.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user"
}
POST /api/instagram/actions/like-story

Like Story

Like a user's Instagram story.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user"
}
POST /api/instagram/actions/comment-story

Comment on Story

Comment on a user's Instagram story.

Request Body:

{
    "device_id": "device123",
    "username": "user123",
    "target_username": "target_user",
    "comment": "👏"  // Optional, defaults to "👏"
}

Task Endpoints

GET /api/instagram/tasks/{task_id}

Get Task Status

Get the status of a specific task.

Response Example:

{
    "success": true,
    "task": {
        "task_id": "task123",
        "task_type": "like_post",
        "status": "completed",
        "created_at": "2024-01-30T12:00:00Z",
        "completed_at": "2024-01-30T12:01:00Z",
        "result": "Post liked successfully"
    }
}
GET /api/instagram/tasks

List All Tasks

Get a list of all tasks and their status.

Response Codes

Success Codes

  • 200 Request successful
  • 201 Resource created

Error Codes

  • 400 Bad request
  • 401 Unauthorized
  • 404 Resource not found
  • 500 Server error
